# Scripts to build ffmpeg for Unix/Android

#### 1. Build for Unix like system, such as Linux, Mac etc.

```shell
$./build-ffmpeg-for-unix.sh
```

#### 2. Build for Android

```
$ export ANDROID_NDK=/opt/android-ndk-r17-beta2
$./build-ffmpeg-for-android.sh
```
